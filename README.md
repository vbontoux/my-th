My-TreasureHunt
===============

This project can be ran using **NPM or Yarn**

## NPM
### Getting Started
You need first to install all dependencies :

`npm install`

### Running the server
You can simpli run the server using the command
`npm start`


## Yarn
### Getting Started
You need first to install all dependencies :

`yarn install`

### Running the server
You can simpli run the server using the command
`yarn start`
